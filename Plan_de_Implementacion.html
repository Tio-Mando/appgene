Plan de Implementación - appgene (App de Oftalmología)

Este documento detalla el diseño y la estructura para construir appgene, una aplicación web diseñada específicamente para optimizar la consulta oftalmológica, enfocada en la gestión de historias clínicas, agenda inteligente de citas/cirugías sin solapamientos y envío simplificado de recetas/mensajes.

Características Principales Propuestas

1. Ficha Médica Oftalmológica Especializada

A diferencia de un historial clínico general, la ficha de oftalmología requiere campos de medición óptica específicos:

  • Agudeza Visual (AV): Lejana y cercana, con y sin corrección, para Ojo Derecho (OD) y Ojo Izquierdo (OS).
  • Refracción: Esfera, Cilindro, Eje, Adición, y Distancia Pupilar.
  • Presión Intraocular (PIO): Crucial para el descarte y control de glaucoma (OD/OS).
  • Sección de Diagnóstico: Observaciones de párpados, córnea, cristalino, y fondo de ojo.

2. Agenda Inteligente Sin Solapamientos

Un motor de validación interno que asegura que:

  • Eventos Bloqueantes: Si se registra una Cirugía en un rango de fecha y hora, el sistema bloqueará ese horario impidiendo agendar citas comunes.
  • Citas de Consulta: Validaciones para que no coincidan dos pacientes a la misma hora exacta si se especifica que el canal de atención es único.
  • Código Visual: Clasificación inmediata mediante colores (por ejemplo, Azul para consulta de primera vez, Verde para controles, Naranja/Rojo para cirugías).

3. Generador de Mensajes y Recetas Clínicas

Un editor donde la doctora escribe el tratamiento y recomendaciones:

Saludos, [Nombre del Paciente] En la cita de hoy se te evaluó: [Diagnóstico/Notas] Recuerda que debes comprar: [Lista de medicamentos/insumos] Nos vemos el [Fecha de Próxima Cita] para tu siguiente consulta.
  • Botón "Enviar por WhatsApp": Abre directamente una pestaña de WhatsApp Web o la aplicación con el número del paciente y el texto pre-cargado para enviarse con un solo clic.

Arquitectura de la Aplicación

Despliegue Rápido y Seguro:
La aplicación se ha desarrollado como una Single Page Application (SPA) autogestionada usando HTML5 Semántico, CSS Premium y Javascript (ES6). Los datos se almacenan de manera local y segura en el navegador mediante localStorage, garantizando privacidad absoluta de las historias médicas del paciente.

Plan de Verificación

  • Prueba de Colisión de Agenda: Intentar programar una consulta en el mismo horario que una cirugía bloqueada. El sistema deniega el registro e indica el conflicto.
  • Prueba de Generación de WhatsApp: Validar que los campos se formateen automáticamente en la plantilla para su envío inmediato.
